### Elasticsearch

Elasticsearch 是一个基于 JSON 的分布式搜索和分析引擎。

### Logstash

Logstash 是动态数据收集管道，拥有可扩展的插件生态系统。

### Kibana

Kibana 可谓 Elastic Stack 的窗户。探索数据并管理堆栈。
